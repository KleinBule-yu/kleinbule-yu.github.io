umask

概述

在Linux系统中，我们创建一个新的文件或者目录，这些新的文件或目录都会有默认权限，umask命令与文件和目录的默认访问有关

核心逻辑

因为目录与文件的默认权限是不一样的，x权限对于目录是非常重要的，必须有x权限你才能进入目录，但是一般文件的建立是不应该有x的权限，都是通常用于数据的记录，不需要x的权限
所以文件的最大权限为-rw-rw-rw-（666），目录的最大权限为drwxrwxrwx（777）

umask默认值

umask的默认值为0022，第一个数字对应的是特殊权限位（SUID，SGID，SBIT），后三个对应着所有者，所属组，其他人
第一位数字（特殊权限屏蔽）：0不屏蔽所有特殊权限，1屏蔽SBIT，2屏蔽SGID，4屏蔽SUID，7屏蔽所有特殊权限
第二位数字（所有者）：通常为0（不屏蔽自己）
第三位数字（所属组）：2代表屏蔽写权限
第四位数字（其他人）：2代表屏蔽写权限
所以文件的默认权限就是666 - 022 = 644 (rw-r–r–)，目录的默认权限是777 - 022 = 755 (rwxr-xr-x)

chattr

概述

chattr命令是change file attributes on a Linux file system的缩写，主要用于设置文件的隐藏权限，改变文件属性的命令，提升文件安全性

参数详解

参数 作用
-i 文件或目录不能删除、修改、写入等全部操作(连BOSS的root也不可以)
-a 只能追加，不能删除、修改，并且非root用户不能操作(只增不减)
-s 会将数据同步写入到磁盘
-c 自动解压缩：读取文件时自动解压缩
-A 锁定文件访问时间，读取文件时不更新访问时间记录
-d 只查看当前目录本身,相当于Is的-d选项
-a 只能追加内容，不能删除和修改就内容
-R 递归处理，将属性应用到当前目录下的文件或者目录

lattr

概述

lsattr 命令用于显示文件的隐藏权限，格式为“Isattr [参数] 文件”。在 Linux系统中，文件的隐藏权限必须使用Isattr 命令来查看，平时使用的ls之类的命令则看不出端倪

参数详解

参数 作用
-a 显示所有文件（包括隐藏文件）
-d 仅显示目录本身
-R 递归显示
