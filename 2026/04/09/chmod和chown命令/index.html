chmod和chown命令

xiao yu2026-04-092026-04-09

概述

在Linux系统中chmod和chown命令都是用来设置权限的，但是它们也是有区别的。chmod是用来设置目录或文件权限的，chown是用来设置更改目录或文件的所有者和所属组的

chmod

使用对象

ugos：表示权限设置所针对的用户类别
u（user）：表示文件或目录的属主（所有者）
g（group）：表示属组内的用户
o（others）：表示其他用户
a（all）：表示所有用户（即u+g+o）

操作动作

+ 或 - 或 =，表示设置权限的操作动作
+：表示添加某个权限
-：表示取消某个权限
=：表示只赋予给定的权限，并取消原有的权限

参数详解

参数 作用
-c 若修改文件或目录后的权限与修改前不一致，才会报告权限更改的动作
-f 若修改文件或目录权限报错，也不要显示错误信息
-v 显示修改文件或目录权限过程中的详细信息
-R 递归对修改目录下的所有文件及子目录一并修改权限
–help 显示chmod说明信息
–version 显示chmod版本信息

chown

参数解释

指令 作用
-c 或 –changes 若修改文件或目录后的权限与修改前不一致，才会报告权限更改的动作
-f 或 –quite 或 –silent 若修改文件或目录权限报错，也不要显示错误信息
-h 或 –no-dereference 显示修改文件或目录权限过程中的详细信息
-R 或 –recursive 递归对修改目录下的所有文件及子目录一并修改权限
-v 或 –verbose 显示chmod说明信息
–dereference 显示chmod版本信息
–help 显示chown说明信息
–reference 把指定文件的属主和属组作为参考，复制给目标文件
–version 显示chown版本信息
