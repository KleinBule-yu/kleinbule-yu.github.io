    у╝║уѓ╣№╝џжџЙС╗ЦУ┐ЂуД╗№╝їТІиУ┤ЮтЄатЇЂућџУЄ│СИіуЎЙ GB уџётЇЋТќЄС╗Х№╝їт»╣тГўтѓеС╗ІУ┤еуџёТќЄС╗Ху│╗у╗ЪТа╝т╝Ј№╝ѕFAT32т░▒тЈ»УЃйСИЇУАї№╝ЅТюЅСИЦТа╝УдЂТ▒ѓ
  • т░єУЎџТІЪуБЂуЏўТІєтѕєТѕљтцџСИфТќЄС╗Х№╝џVMwareС╝џТіі20GBуБЂуЏўтѕЄТѕљтцџСИфу║д4GBСйюућеуџёт░ЈТќЄС╗ХтГўТћЙтюеТќЄС╗Хтц╣жЄї№╝ѕD:\rocky10.1\rocky10.1-s001.vmdk№╝ї-s002.vmdkС╗ЦтЈіу▒╗Тје№╝Ѕ
    С╝ўуѓ╣№╝џТќ╣СЙ┐уД╗тіетњїтцЄС╗й№╝їТЃ│ТііУЎџТІЪТю║ТІиУ┤Ютѕ░ U уЏўТѕќуД╗тіеуАгуЏў№╝їтЇЋСИфУХЁтцДТќЄС╗Х№╝ѕТ»ћтдѓ 100GB№╝ЅтюеТЌДТа╝т╝ЈуБЂуЏў№╝ѕтдѓ FAT32№╝ЅСИіТЌаТ│ЋтГўтѓе№╝їСИћТІиУ┤ЮтцДТќЄС╗Хт«╣ТўЊтЄ║жћЎсђѓТІєтѕєТѕљт░ЈТќЄС╗ХТЏ┤уе│т«џ
    у╝║уѓ╣№╝џТђДУЃйУЙЃти«№╝їУ»╗тєЎтцДжЄЈТЋ░ТЇ«ТЌХ№╝їуБЂтц┤ТѕќТќЄС╗Ху┤бт╝ЋжюђУдЂтюетцџСИфт░ЈТќЄС╗ХжЌ┤тѕЄТЇб№╝їуљєУ«║СИіТђДУЃйТ»ћтЇЋТќЄС╗ХуеЇти«

Та╣ТЇ«СИфС║║жюђТ▒ѓжђЅТІЕ№╝їтюеУ┐ЎжЄїТѕЉжђЅт░єУЎџТІЪуБЂуЏўтГўтѓеСИ║тЇЋСИфТќЄС╗Х№╝їуѓ╣тЄ╗СИІСИђТГЦ

уѓ╣тЄ╗УЄфт«џС╣ЅуАгС╗Х

тєЁтГўтњїтцёуљєтЎеТа╣ТЇ«жюђТ▒ѓТЮЦ№╝їТѕЉжђЅТІЕж╗ўУ«цт░▒УАї№╝їуёХтљјуѓ╣тЄ╗РђюТќ░ CD/DVD№╝ѕIDE№╝ЅРђЮ - РђюСй┐уће ISOТўатЃЈТќЄС╗ХРђЮ - РђюТхЈУДѕРђЮ жђЅТІЕСйажЋютЃЈтГўТћЙуџёСйЇуй«

уА«У«цтЦйС┐АТЂ»С╣Ітљјт░▒тЈ»С╗Цуѓ╣тЄ╗т«їТѕљС║є

уёХтљјт░▒тЈ»С╗Цт╝ђтљ»УЎџТІЪТю║С║є№╝їуѓ╣тЄ╗Рђют╝ђтљ»ТГцУЎџТІЪТю║РђЮ

У┐ЏтЁЦу│╗у╗Ът«ЅУБЁт╝Ћт»╝уЋїжЮб№╝їrocky10.1У┐ЏтЁЦС╣Ітљјж╗ўУ«цТў»жђЅТІЕуггС║їСИфРђюTest this Media & install Rocky Linux 10.1РђЮ№╝їуѓ╣тЄ╗СИіТќ╣тљЉжћ«жђЅТІЕРђюInstall Rocky Linux 10.1РђЮ№╝їТїЅСИІтЏъУйд

Test this Media & install Rocky Linux 10.1№╝џтюеУ┐ЏтЁЦт«ЅУБЁуЋїжЮбС╣ІтЅЇ№╝їу│╗у╗ЪС╝џтЁѕт»╣СйаТїѓУййуџё ISO жЋютЃЈУ┐ЏУАїСИђТгАтЁеуЏўТЅФТЈЈ№╝їТаИт»╣ТаАжфїуаЂ№╝ѕChecksum№╝Ѕ

У┐ЏтЁЦLinuxу│╗у╗ЪтљјтЈ░ТюЇтіАуџётіаУййУ»дТЃЁуЋїжЮб№╝їтљ»тіе№╝ѕТѕќтЁ│жЌГ/жЄЇтљ»№╝ЅУ┐ЄуеІСИГжЃйУЃйуюІтѕ░№╝їуГЅтЙЁСИђСИІт░▒тЈ»С╗ЦУи│тѕ░У»ГУеђжђЅТІЕуЋїжЮб

тдѓТъюжђЅТІЕуџёТў»уггС║їСИф№╝їт«ЃС╝џтЄ║уј░У┐Џт║дТЮА№╝їтЈ»С╗ЦСй┐ућеescжћ«тіажђЪУ»дТЃЁуЋїжЮб

тѕ░С║єСИІжЮбУ┐ЎСИфуЋїжЮбт░▒тЈ»С╗ЦСИЇућеТїЅesqжћ«С║є

жђЅТІЕу│╗у╗ЪУ»ГУеђ№╝їТа╣ТЇ«СИфС║║жюђТ▒ѓТЮЦ№╝їТѕЉжђЅТІЕуџёТў»УІ▒У»Г№╝їуёХтљјуѓ╣тЄ╗Continue№╝ѕу╗Ду╗Г№╝Ѕ

уѓ╣тЄ╗РђюInstallation DestinationРђЮ№╝ѕт«ЅУБЁуЏ«ТаЄСйЇуй«№╝Ѕ№╝їт░▒Тў»ТііLinuxт«ЅУБЁтѕ░тЊфтЮЌуАгуЏў№╝їтдѓСйЋтѕњтѕєуБЂуЏўуЕ║жЌ┤№╝їт░▒Тў»тюежђЅТІЕу│╗у╗ЪуЏў

ж╗ўУ«цуБЂуЏўТў»тІЙжђЅСИі№╝їуЏ┤ТјЦуѓ╣Done
Storage Configuration№╝ѕтГўтѓежЁЇуй«№╝Ѕ№╝їж╗ўУ«цжђЅТІЕAutomatic№╝ѕУЄфтіе№╝Ѕ№╝їТёЈТђЮТў»у│╗у╗ЪС╝џУЄфтіеУ┐ЏУАїтѕєтї║№╝їжђѓтљѕТќ░ТЅІ

тЈфУдЂУГдтЉіТХѕтц▒С║єт░▒т╝ётЦйС║є

жђЅТІЕSoftware Selection№╝ѕУй»С╗ХжђЅТІЕ№╝Ѕ
ж╗ўУ«цт░▒Тў»Server with GUI№╝ѕтИдтЏЙтйбуЋїжЮбуџёТюЇтіАтЎе№╝Ѕ№╝їС╣Ът░▒Тў»тЏЙтйбтїќуЋїжЮб

Server with GUI№╝ѕтИдтЏЙтйбуЋїжЮбуџёТюЇтіАтЎе№╝Ѕ№╝џТюЅж╝аТаЄтЈ»С╗Цуѓ╣№╝їтИдТАїжЮб№╝ѕжђџтИИТў» GNOME№╝Ѕ№╝їжђѓтљѕтѕЮтГдУђЁ
Server№╝ѕТюЇтіАтЎе№╝Ѕ№╝џТ▓АТюЅТАїжЮб№╝їу║»тЉйС╗цУАї№╝їСйєжбёУБЁС║єтИИућеуџёТюЇтіАтЎеу«АуљєтиЦтЁи
Minimal Install№╝ѕТюђт░Јтїќт«ЅУБЁ№╝Ѕ№╝џТюђУіѓуюЂУхёТ║љсђѓ тЈфт«ЅУБЁТюђТаИт┐ЃуџёУ┐љУАїу╗ёС╗Х№╝їжђѓтљѕтюеУ┐йТ▒ѓТъЂУЄ┤ТђДУЃйсђЂжФўт«ЅтЁеТђДуџёућЪС║Дуј»тбЃ№╝їТѕќУђЁТ»ћУхЏСИГУдЂТ▒ѓС╗јжЏХт╝ђтДІжЁЇуй«уџёТЃЁтєх
Workstation№╝ѕтиЦСйюуФЎ№╝Ѕ№╝џу▒╗С╝╝СИфС║║ућхУёЉ№╝їтИдТюЅтцДжЄЈуџётіътЁгУй»С╗ХсђЂТњГТћЙтЎеуГЅ
Virtualization Host№╝ѕУЎџТІЪтїќСИ╗Тю║№╝Ѕ№╝џСИЊжЌеућеТЮЦтЂџТ»ЇТю║№╝їтєЇтюежЄїжЮбт╝ђ KVM УЎџТІЪТю║уџёуј»тбЃ
ТѕЉУ┐ЎжЄїжђЅТІЕТюђт░Јтїќт«ЅУБЁ№╝їтљјжЮбТѕЉС╝џт▒Ћуц║Server with GUIтњїMinimal Installтњїт«ЅУБЁтљјуџётї║тѕФ

уѓ╣тЄ╗РђюRoot AccountРђЮ№╝ѕrootУ┤дТѕи№╝Ѕу╗ЎLinuxуџёrootу«АуљєтЉўУ«Йуй«т»єуаЂ
rootу«АуљєтЉўт░▒Тў»LinuxСИГТЮЃжЎљТюђтцДуџёСИђСИфућеТѕи№╝їт░▒тЦйТ»ћтЂџтЁгтЈИуџёУђЂТЮ┐№╝їУђЂТЮ┐№╝ѕroot№╝ЅТюЅТюђжФўТЮЃжЎљтј╗у«АуљєтЁгтЈИ№╝ѕLinuxу│╗у╗Ъ№╝Ѕ

уѓ╣тЄ╗РђюEnable root accountРђЮ№╝ѕтљ»ућеrootУ┤дТѕи№╝Ѕ№╝їт╣ХУ«Йуй«rootу«АуљєтЉўуџёт»єуаЂ

  • тдѓТъютІЙжђЅС║єуџёУ»Ю№╝їт«ЅУБЁтЦйLinuxС╣Ітљј№╝їтЈ»С╗ЦwindowsСИіТЅЊт╝ђу╗ѕуФ»№╝ѕCMDтњїPowerShell№╝ЅСй┐ућеsshтЉйС╗цУ┐юуеІжЊЙТјЦуЎ╗тйЋLinux№╝ѕСЙІтдѓ ssh root@192.168.100.1№╝Ѕ
  • тдѓТъюСИЇтІЙжђЅуџёУ»Ю№╝їУдЂтюеLinuxуџёsshжЁЇуй«ТќЄС╗Х№╝ѕ/etc/ssh/sshd_config№╝Ѕ№╝їС┐«Тћ╣угг40УАїуџёРђю#PermitRootLogin prohibit-passwordРђЮСИ║РђюPermitRootLogin yesРђЮ№╝їТЅЇтЈ»С╗ЦСй┐ућеу╗ѕуФ»Сй┐ућеsshтЉйС╗цУ┐юуеІжЊЙТјЦуЎ╗тйЋLinux

уѓ╣тЄ╗РђюBegin InstallationРђЮ№╝ѕт╝ђтДІт«ЅУБЁ№╝Ѕ
