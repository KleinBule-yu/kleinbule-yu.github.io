жҳҜеӣ дёәдҪ зҡ„WindowsејҖеҗҜдәҶHyper-VжҲ–иҖ…зӣёе…ізҡ„е®үе…ЁеҠҹиғҪпјҲеҰӮеҶ…ж ёйҡ”зҰ»/йҳІжҠӨпјүпјҢиҝҷдјҡе’Ң VMware зҡ„иҝҗиЎҢжңәеҲ¶дә§з”ҹеҶІзӘҒ

йҖүжӢ©иҷҡжӢҹжңәзҡ„зЈҒзӣҳеӨ§е°Ҹд»ҘеҸҠиҷҡжӢҹжңәзҡ„зЈҒзӣҳж–Үд»¶еңЁдҪ WindowsпјҲзү©зҗҶжңәпјүдёӯж—¶жҳҜд»Ҙд»Җд№ҲеҪўејҸеӯҳеңЁзҡ„
зЈҒзӣҳеӨ§е°Ҹй»ҳи®ӨжҳҜ20GBпјҢж №жҚ®дёӘдәәйңҖжұӮи®ҫзҪ®еӨ§е°ҸпјҲиҝҷдёӘиҷҡжӢҹжңәзЈҒзӣҳеӨ§е°ҸдёҚзӯүдәҺзү©зҗҶзЈҒзӣҳеӨ§е°ҸпјҢдёҚдјҡеҚ з”Ёе®һйҷ…зЈҒзӣҳз©әй—ҙпјү

  • е°ҶиҷҡжӢҹзЈҒзӣҳеӯҳеӮЁдёәеҚ•дёӘж–Үд»¶пјҡVMwareдјҡеңЁдҪ зҡ„ж–Үд»¶еӨ№дёҠеҲӣе»әдёҖдёӘе·ЁеӨ§зҡ„.vmdkж–Үд»¶пјҲD:\rocky10.1\rocky10.1.vmdkпјү
    дјҳзӮ№пјҡжҖ§иғҪиҫғеҘҪпјҢеңЁж•°жҚ®еӯҳеӮЁеңЁзү©зҗҶзЈҒзӣҳдёҠзӣёеҜ№иҝһз»ӯпјҢиҜ»еҶҷж•ҲзҺҮжӣҙй«ҳ
    зјәзӮ№пјҡйҡҫд»ҘиҝҒз§»пјҢжӢ·иҙқеҮ еҚҒз”ҡиҮідёҠзҷҫ GB зҡ„еҚ•ж–Үд»¶пјҢеҜ№еӯҳеӮЁд»ӢиҙЁзҡ„ж–Үд»¶зі»з»ҹж јејҸпјҲFAT32е°ұеҸҜиғҪдёҚиЎҢпјүжңүдёҘж јиҰҒжұӮ
  • е°ҶиҷҡжӢҹзЈҒзӣҳжӢҶеҲҶжҲҗеӨҡдёӘж–Үд»¶пјҡVMwareдјҡжҠҠ20GBзЈҒзӣҳеҲҮжҲҗеӨҡдёӘзәҰ4GBдҪңз”Ёзҡ„е°Ҹж–Үд»¶еӯҳж”ҫеңЁж–Үд»¶еӨ№йҮҢпјҲD:\rocky10.1\rocky10.1-s001.vmdkпјҢ-s002.vmdkд»ҘеҸҠзұ»жҺЁпјү
    дјҳзӮ№пјҡж–№дҫҝз§»еҠЁе’ҢеӨҮд»ҪпјҢжғіжҠҠиҷҡжӢҹжңәжӢ·иҙқеҲ° U зӣҳжҲ–з§»еҠЁзЎ¬зӣҳпјҢеҚ•дёӘи¶…еӨ§ж–Үд»¶пјҲжҜ”еҰӮ 100GBпјүеңЁж—§ж јејҸзЈҒзӣҳпјҲеҰӮ FAT32пјүдёҠж— жі•еӯҳеӮЁпјҢдё”жӢ·иҙқеӨ§ж–Үд»¶е®№жҳ“еҮәй”ҷгҖӮжӢҶеҲҶжҲҗе°Ҹж–Үд»¶жӣҙзЁіе®ҡ
    зјәзӮ№пјҡжҖ§иғҪиҫғе·®пјҢиҜ»еҶҷеӨ§йҮҸж•°жҚ®ж—¶пјҢзЈҒеӨҙжҲ–ж–Үд»¶зҙўеј•йңҖиҰҒеңЁеӨҡдёӘе°Ҹж–Үд»¶й—ҙеҲҮжҚўпјҢзҗҶи®әдёҠжҖ§иғҪжҜ”еҚ•ж–Үд»¶зЁҚе·®

ж №жҚ®дёӘдәәйңҖжұӮйҖүжӢ©пјҢеңЁиҝҷйҮҢжҲ‘йҖүе°ҶиҷҡжӢҹзЈҒзӣҳеӯҳеӮЁдёәеҚ•дёӘж–Үд»¶пјҢзӮ№еҮ»дёӢдёҖжӯҘ

зӮ№еҮ»иҮӘе®ҡд№үзЎ¬д»¶

еҶ…еӯҳе’ҢеӨ„зҗҶеҷЁж №жҚ®йңҖжұӮжқҘпјҢжҲ‘йҖүжӢ©й»ҳи®Өе°ұиЎҢпјҢз„¶еҗҺзӮ№еҮ»вҖңж–° CD/DVDпјҲIDEпјүвҖқ - вҖңдҪҝз”Ё ISOжҳ еғҸж–Үд»¶вҖқ - вҖңжөҸи§ҲвҖқ йҖүжӢ©дҪ й•ңеғҸеӯҳж”ҫзҡ„дҪҚзҪ®

зЎ®и®ӨеҘҪдҝЎжҒҜд№ӢеҗҺе°ұеҸҜд»ҘзӮ№еҮ»е®ҢжҲҗдәҶ

з„¶еҗҺе°ұеҸҜд»ҘејҖеҗҜиҷҡжӢҹжңәдәҶпјҢзӮ№еҮ»вҖңејҖеҗҜжӯӨиҷҡжӢҹжңәвҖқ

иҝӣе…Ҙзі»з»ҹе®үиЈ…еј•еҜјз•ҢйқўпјҢrocky10.1иҝӣе…Ҙд№ӢеҗҺй»ҳи®ӨжҳҜйҖүжӢ©з¬¬дәҢдёӘвҖңTest this Media & install Rocky Linux 10.1вҖқпјҢзӮ№еҮ»дёҠж–№еҗ‘й”®йҖүжӢ©вҖңInstall Rocky Linux 10.1вҖқпјҢжҢүдёӢеӣһиҪҰ
иҝҷдёӨдёӘеҢәеҲ«дёҚеӨ§пјҢз¬¬дёҖдёӘжӣҙеҝ«дёҖзӮ№
Install Rocky Linux 10.1пјҡз«ӢеҚіеҗҜеҠЁе®үиЈ…зЁӢеәҸпјҢи·іиҝҮеҜ№е®үиЈ…д»ӢиҙЁпјҲISO ж–Үд»¶пјүзҡ„е®Ңж•ҙжҖ§жЈҖжҹҘ
Test this Media & install Rocky Linux 10.1пјҡеңЁиҝӣе…Ҙе®үиЈ…з•Ңйқўд№ӢеүҚпјҢзі»з»ҹдјҡе…ҲеҜ№дҪ жҢӮиҪҪзҡ„ ISO й•ңеғҸиҝӣиЎҢдёҖж¬Ўе…Ёзӣҳжү«жҸҸпјҢж ёеҜ№ж ЎйӘҢз ҒпјҲChecksumпјү

иҝӣе…ҘLinuxзі»з»ҹеҗҺеҸ°жңҚеҠЎзҡ„еҠ иҪҪиҜҰжғ…з•ҢйқўпјҢеҗҜеҠЁпјҲжҲ–е…ій—ӯ/йҮҚеҗҜпјүиҝҮзЁӢдёӯйғҪиғҪзңӢеҲ°пјҢзӯүеҫ…дёҖдёӢе°ұеҸҜд»Ҙи·іеҲ°иҜӯиЁҖйҖүжӢ©з•Ңйқў

еҰӮжһңйҖүжӢ©зҡ„жҳҜз¬¬дәҢдёӘпјҢе®ғдјҡеҮәзҺ°иҝӣеәҰжқЎпјҢеҸҜд»ҘдҪҝз”Ёescй”®еҠ йҖҹиҜҰжғ…з•Ңйқў

еҲ°дәҶдёӢйқўиҝҷдёӘз•Ңйқўе°ұеҸҜд»ҘдёҚз”ЁжҢүesqй”®дәҶ

йҖүжӢ©зі»з»ҹиҜӯиЁҖпјҢж №жҚ®дёӘдәәйңҖжұӮжқҘпјҢжҲ‘йҖүжӢ©зҡ„жҳҜиӢұиҜӯпјҢз„¶еҗҺзӮ№еҮ»ContinueпјҲз»§з»ӯпјү

зӮ№еҮ»вҖңInstallation DestinationвҖқпјҲе®үиЈ…зӣ®ж ҮдҪҚзҪ®пјүпјҢе°ұжҳҜжҠҠLinuxе®үиЈ…еҲ°е“Әеқ—зЎ¬зӣҳпјҢеҰӮдҪ•еҲ’еҲҶзЈҒзӣҳз©әй—ҙпјҢе°ұжҳҜеңЁйҖүжӢ©зі»з»ҹзӣҳ

й»ҳи®ӨзЈҒзӣҳжҳҜеӢҫйҖүдёҠпјҢзӣҙжҺҘзӮ№Done
Storage ConfigurationпјҲеӯҳеӮЁй…ҚзҪ®пјүпјҢй»ҳи®ӨйҖүжӢ©AutomaticпјҲиҮӘеҠЁпјүпјҢж„ҸжҖқжҳҜзі»з»ҹдјҡиҮӘеҠЁиҝӣиЎҢеҲҶеҢәпјҢйҖӮеҗҲж–°жүӢ

еҸӘиҰҒиӯҰе‘Ҡж¶ҲеӨұдәҶе°ұеј„еҘҪдәҶ

йҖүжӢ©Software SelectionпјҲиҪҜд»¶йҖүжӢ©пјү
й»ҳи®Өе°ұжҳҜServer with GUIпјҲеёҰеӣҫеҪўз•Ңйқўзҡ„жңҚеҠЎеҷЁпјүпјҢд№ҹе°ұжҳҜеӣҫеҪўеҢ–з•Ңйқў

Server with GUIпјҲеёҰеӣҫеҪўз•Ңйқўзҡ„жңҚеҠЎеҷЁпјүпјҡжңүйј ж ҮеҸҜд»ҘзӮ№пјҢеёҰжЎҢйқўпјҲйҖҡеёёжҳҜ GNOMEпјүпјҢйҖӮеҗҲеҲқеӯҰиҖ…
ServerпјҲжңҚеҠЎеҷЁпјүпјҡжІЎжңүжЎҢйқўпјҢзәҜе‘Ҫд»ӨиЎҢпјҢдҪҶйў„иЈ…дәҶеёёз”Ёзҡ„жңҚеҠЎеҷЁз®ЎзҗҶе·Ҙе…·
Minimal InstallпјҲжңҖе°ҸеҢ–е®үиЈ…пјүпјҡжңҖиҠӮзңҒиө„жәҗгҖӮ еҸӘе®үиЈ…жңҖж ёеҝғзҡ„иҝҗиЎҢз»„д»¶пјҢйҖӮеҗҲеңЁиҝҪжұӮжһҒиҮҙжҖ§иғҪгҖҒй«ҳе®үе…ЁжҖ§зҡ„з”ҹдә§зҺҜеўғпјҢжҲ–иҖ…жҜ”иөӣдёӯиҰҒжұӮд»Һйӣ¶ејҖе§Ӣй…ҚзҪ®зҡ„жғ…еҶө
WorkstationпјҲе·ҘдҪңз«ҷпјүпјҡзұ»дјјдёӘдәәз”өи„‘пјҢеёҰжңүеӨ§йҮҸзҡ„еҠһе…¬иҪҜд»¶гҖҒж’ӯж”ҫеҷЁзӯү
Virtualization HostпјҲиҷҡжӢҹеҢ–дё»жңәпјүпјҡдё“й—Ёз”ЁжқҘеҒҡжҜҚжңәпјҢеҶҚеңЁйҮҢйқўејҖ KVM иҷҡжӢҹжңәзҡ„зҺҜеўғ
жҲ‘иҝҷйҮҢйҖүжӢ©жңҖе°ҸеҢ–е®үиЈ…пјҢеҗҺйқўжҲ‘дјҡеұ•зӨәServer with GUIе’ҢMinimal Installе’Ңе®үиЈ…еҗҺзҡ„еҢәеҲ«

зӮ№еҮ»вҖңRoot AccountвҖқпјҲrootиҙҰжҲ·пјүз»ҷLinuxзҡ„rootз®ЎзҗҶе‘ҳи®ҫзҪ®еҜҶз Ғ
rootз®ЎзҗҶе‘ҳе°ұжҳҜLinuxдёӯжқғйҷҗжңҖеӨ§зҡ„дёҖдёӘз”ЁжҲ·пјҢе°ұеҘҪжҜ”еҒҡе…¬еҸёзҡ„иҖҒжқҝпјҢиҖҒжқҝпјҲrootпјүжңүжңҖй«ҳжқғйҷҗеҺ»з®ЎзҗҶе…¬еҸёпјҲLinuxзі»з»ҹпјү

зӮ№еҮ»вҖңEnable root accountвҖқпјҲеҗҜз”ЁrootиҙҰжҲ·пјүпјҢе№¶и®ҫзҪ®rootз®ЎзҗҶе‘ҳзҡ„еҜҶз Ғ
зӮ№еҮ»еӢҫйҖүвҖңAllow root SSH login with passwordвҖқ пјҲе…Ғи®ёrootз”ЁжҲ·дҪҝз”ЁеҜҶз ҒиҝӣиЎҢSSHзҷ»еҪ•пјү

  • еҰӮжһңеӢҫйҖүдәҶзҡ„иҜқпјҢе®үиЈ…еҘҪLinuxд№ӢеҗҺпјҢеҸҜд»ҘwindowsдёҠжү“ејҖз»Ҳз«ҜпјҲCMDе’ҢPowerShellпјүдҪҝз”Ёsshе‘Ҫд»ӨиҝңзЁӢй“ҫжҺҘзҷ»еҪ•LinuxпјҲдҫӢеҰӮ ssh root@192.168.100.1пјү
  • еҰӮжһңдёҚеӢҫйҖүзҡ„иҜқпјҢиҰҒеңЁLinuxзҡ„sshй…ҚзҪ®ж–Үд»¶пјҲ/etc/ssh/sshd_configпјүпјҢдҝ®ж”№з¬¬40иЎҢзҡ„вҖң#PermitRootLogin prohibit-passwordвҖқдёәвҖңPermitRootLogin yesвҖқпјҢжүҚеҸҜд»ҘдҪҝз”Ёз»Ҳз«ҜдҪҝз”Ёsshе‘Ҫд»ӨиҝңзЁӢй“ҫжҺҘзҷ»еҪ•Linux

еҸӘиҰҒи®ҫзҪ®еҘҪrootеҜҶз Ғд№ӢеҗҺпјҢUSER SETTINGSпјҲз”ЁжҲ·и®ҫзҪ®пјүе°ұдёҚдјҡжңүиӯҰе‘ҠдәҶ
дёӢйқўзҡ„User Creationе°ұжҳҜз”ЁжқҘеҲӣе»әжҷ®йҖҡз”ЁжҲ·пјҢж №жҚ®дёӘдәәйңҖжұӮпјҢеҲӣдёҚеҲӣе»әйғҪеҸҜд»ҘгҖӮжҷ®йҖҡз”ЁжҲ·е°ұжІЎжңүrootжқғйҷҗиҝҷд№ҲеӨ§пјҢж¶үеҸҠеҲ°зі»з»ҹз®ЎзҗҶзҡ„е°ұжІЎжңүжқғйҷҗеҺ»жү§иЎҢпјҢж—Ҙеёёзҡ„е‘Ҫд»ӨжқғйҷҗиҝҳжҳҜеҸҜд»Ҙзҡ„пјҢе°ұжҜ”еҸҜд»ҘжҜ”дҪңдёәrootжүӢдёӢзҡ„е‘ҳе·Ҙеҗ§
зӮ№еҮ»вҖңBegin InstallationвҖқпјҲејҖе§Ӣе®үиЈ…пјү

з„¶еҗҺе°ұзӯүеҫ…иҷҡжӢҹжңәе®үиЈ…е®ҢжҜ•е°ұиЎҢ

е®үиЈ…е®ҢжҜ•д№ӢеҗҺпјҢзӮ№еҮ»вҖңReboot SystemвҖқпјҲйҮҚеҗҜзі»з»ҹпјү

е‘Ҫд»ӨиЎҢзҡ„иҜқе°ұжҳҜдёҖзүҮй»‘иүІпјҢз„¶еҗҺжңүеҮәзҺ°localhost loginпјҡзҡ„еӯ—ж ·жҳҜи®©дҪ иҫ“е…ҘиҙҰеҸ·зҡ„пјҢиҫ“е…Ҙе®ҢиҙҰеҸ·еӣһиҪҰд№ӢеҗҺе°ұжҳҜиҫ“е…ҘеҜҶз Ғ

еӣҫеҪўеҢ–з•Ңйқўзҡ„иҜқе°ұзұ»дјјдёҺwindowsжңүејҖжңәеҠЁз”»еҲ°дёӢйқўзҡ„йЎөйқўзӮ№еҮ»вҖңStart SetupвҖқпјҲејҖе§Ӣи®ҫзҪ®пјү

Location ServicesжҳҜејҖеҗҜең°зҗҶдҪҚзҪ®жңҚеҠЎзҡ„пјҢзӮ№еҮ»Next

и®ҫзҪ®з”ЁжҲ·зҡ„еҜҶз ҒпјҢз„¶еҗҺзӮ№Next
