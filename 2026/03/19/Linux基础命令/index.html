Vim 编辑器

Vim 编辑器 是 Linux 系统中经典的 vi 编辑器的增强版本。它具有三种核心模式：命令模式、输入模式、末行模式。

1. 命令模式

用于移动光标、复制、粘贴、删除等核心操作。

指令 作用与说明
dd 删除当前行（前面加数字可指定删除多行）
yy 复制当前行（前面加数字可指定复制多行）
p 粘贴内容
u 或 Ctrl+r 撤销或重做
v/V/Ctrl+v 进入可视模式（进行字符、行或块选择）
i/a/o 切换至输入模式
: 进入末行模式

2. 输入模式

直接输入或编辑文本内容。

按键 插入位置
i 光标前插入
a 光标后插入
I 行首插入
A 行尾插入
o 下方新建一行插入
O 上方新建一行插入
ESC 返回命令模式

3. 末行模式

在命令模式下按 `:` 进入，执行文件保存、退出等操作。

指令 作用
:w 保存文件
:q 退出 Vim
:wq 或 :x 保存并退出
:q! 强制退出（不保存修改）
:s/old/new 替换当前行第一个匹配项
:%s/old/new/g 全局替换所有匹配项
:set number 显示行数

编辑文件，输入vim+文件名

进入配置页面，此时我们正在命令模式

需要编辑可以使用i/a/o等按键进入插入，按照具体情况根据需求使用；输入i后，左下角变为INSERT，成功进入插入模式，此时可以随意写入内容

写入内容后，按ESC键返回命令模式，然后输入yy，就可以复制光标所在那一行的内容

按下p，将刚刚复制的内容粘贴

按住shift键，输入：，进入末行模式

输入set number，回车，显示行数

输入:wq!强制保存并退出，此时文件编辑成功
